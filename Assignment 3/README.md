# Assignment 3 - Binary Search Tree
Implements a binary search tree (BST) as a class template and using dynamic node and tree classes. Steps:
1. Initialises BST with student names and grades which are printed with inorder traversal.
2. Number of students with grade < 50 is printed out
3. The average grade is printed out
4. All students with a grade < 50 are deleted from the BST
5. Steps  2 and 3 are repeated
6. Searches for an input name in the tree and prints the name and grade if found.
